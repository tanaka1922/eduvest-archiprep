# 二級建築士 学科試験対策アプリ 仕様書

## バージョン
- 作成日: 2026-02-14
- バージョン: v1.0
- コードネーム: **ArchiPrep 2級**
- ベースモデル: KIT構造力学アプリ v4.2

---

## 概要

2026年度（令和8年度）二級建築士学科試験の全4科目をカバーするHTML形式の学習アプリ。
生徒向け教材として、偏差値30-50レベルの学習者にも理解できる丁寧な解説を提供する。

---

## 試験情報（リサーチ結果）

### 試験構成
| 科目 | 問題数 | 時間 | 合格基準 |
|------|--------|------|----------|
| 学科I 建築計画 | 25問 | 3時間（IとII合わせて） | 各科目13点以上 |
| 学科II 建築法規 | 25問 | ↑ | ↑ |
| 学科III 建築構造 | 25問 | 3時間（IIIとIV合わせて） | ↑ |
| 学科IV 建築施工 | 25問 | ↑ | ↑ |
| **合計** | **100問** | **6時間** | **総合60点以上** |

※ 各科目25問 × 4肢択一 = 合計100問
※ 法規のみ法令集持込可

### 2025年度試験の最新傾向（日建学院・総合資格の分析）
- 過去問からの出題が中心だが「応用問題」が増加
- 1問に対してより幅広い知識が必要
- 正答肢が初出題の問題は100問中22問（約22%）
- **2025年4月法改正出題あり**：
  - 確認済証の交付
  - 構造計算の規模要件（「高さ13m以下かつ軒高9m以下」→「階数3以下かつ高さ16m以下」）
  - 建築士法（二級建築士の業務範囲拡大）
  - 建築物省エネ法

## 技術仕様

### アーキテクチャ
- **形式**: 単一HTMLファイル × 科目数（4ファイル + 統合版1ファイル）
- **言語**: HTML5 + CSS3 + JavaScript（バニラ）
- **数式**: MathJax 3（構造力学の計算問題用）
- **フォント**: Noto Sans JP（Google Fonts CDN）
- **レスポンシブ**: モバイルファースト設計

### ファイル構成
```
archiprep-2kyu/
├── index.html          # メニュー画面（科目選択）
├── planning.html       # 学科I 建築計画
├── regulations.html    # 学科II 建築法規
├── structure.html      # 学科III 建築構造
├── construction.html   # 学科IV 建築施工
└── README.md           # プロジェクト説明
```

### KIT構造力学アプリからの継承機能
1. **4択問題形式** — タップで解答 → 即時フィードバック
2. **正解率トラッキング** — セッション内のスコア表示
3. **XP / バッジシステム** — ゲーミフィケーション要素
4. **色分け解説ボックス** — 思考（橙）・計算（青）・ポイント（緑）・注意（赤）
5. **レーベンシュタイン距離** — 穴埋め問題のスペル許容（※法規では不要、選択式のため）
6. **モバイル対応** — スマホで電車内学習

### 新規追加機能
1. **科目別モード** — 4科目を独立して学習可能
2. **分野別フィルタ** — 「構造力学だけ」「建築史だけ」等で絞り込み
3. **弱点分析** — 不正解の分野を自動集計・表示
4. **法改正マーク** — 🆕 マークで2025-2026年法改正問題を明示
5. **条文リンク** — 法規問題で該当条文番号を表示（法令集で引く練習に）
6. **模擬試験モード** — 25問 × 制限時間で本番シミュレーション
7. **シャッフル出題** — 問題順をランダム化

    {"id": "c", "text": "均等拡散面からの光度は...", "correct": false},

## 問題作成ガイドライン

### リサーチソース（優先順）
1. **過去問**: 建築技術教育普及センター公開の過去問（最重要）
2. **法改正情報**: 国土交通省告示・通達（2025-2026年）
3. **TAC/日建/総合資格**: 出題傾向分析レポート
4. **建築基準法**: e-Gov法令検索（最新条文確認）

### 法改正チェックリスト（2025-2026年）
- [ ] 建築基準法: 構造計算規模要件変更（階数3以下かつ高さ16m以下）
- [ ] 建築士法: 二級建築士業務範囲拡大
- [ ] 省エネ法: 建築物省エネ法改正対応
- [ ] 確認済証の交付手続き変更
- [ ] その他2026年1月1日施行分

### 解説の書き方（偏差値30-50対応）
1. **まず結論** — 正解と理由を1行で
2. **なぜ間違いか** — 各選択肢の正誤理由を丁寧に
3. **覚え方のコツ** — 語呂合わせ・イメージ図・比較表
4. **関連知識** — 「これも一緒に覚えよう」で知識を広げる
5. **法規の場合** — 条文番号 + 法令集での探し方ヒント

## 制約条件
1. オフライン動作必須（CDN以外の外部依存なし）
2. 1ファイルあたり最大100KB（モバイル対応）
3. 法規問題は2026年1月1日施行法令に準拠
4. 著作権: 過去問そのものの転載は不可 → 類題・予想問題として作成
5. 解説に図・イラストが必要な場合はSVGで埋め込み

## 受け入れ基準
- [ ] 全4科目のHTMLファイルが正常に表示される
- [ ] モバイル（iPhone SE幅320px）で操作可能
- [ ] MathJaxが正しくレンダリングされる（構造力学）
- [ ] 正解率トラッキングが動作する
- [ ] 法改正問題に🆕マークがついている
- [ ] 200問以上の問題が収録されている（フェーズ1）

## Cowork制作フロー

### Phase 1: リサーチ & 問題作成（DEEPリサーチ）
1. 各科目の頻出テーマを過去問から抽出
2. 2025-2026年法改正の全項目リスト化
3. 科目ごとに50問ずつ予想問題を作成
4. 解説文を偏差値30-50レベルで執筆

### Phase 2: アプリ実装
1. KIT構造力学アプリのHTMLテンプレートを流用
2. 問題JSONデータを組み込み
3. 科目別カラーリング・UI調整
4. 模擬試験モード実装

### Phase 3: 品質管理
1. 法規問題の条文正確性チェック
2. 全選択肢の正誤確認
3. モバイル表示テスト
4. 生徒にβテスト
